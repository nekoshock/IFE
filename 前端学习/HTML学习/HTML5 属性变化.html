HTML5属性变化

2019-2-27 15:34:03

input

input email

input type="email"

终于知道为什么要这么写,虽然在PC端与text的差距表现不出来,但移动端的输入法认识这个,看到type="email"时会启动专用的键盘,键盘上会有@和.com的快捷键.

input url

又叫统一资源定位符

input type="url"

与email一样,外观无变化,仍然是给移动端的键盘看的.键盘上会有.com之类的按键.

input tel

电话号码 input type="tel"

与之前的都一样,移动端键盘会给出电话的数字输入法,会有*#之类的按键.

input type="number"

数字类型,框上有上下两个成对箭头,框中只认数字和数字符号(如常数e)

input的date pickers

获取年月日 input type="date"

会唤醒一个日历的样式,来获取一个日期.

获取年月 input type="month"

type="date"的阉割版,只剩年月可以选.

获取时间 input type="time"

唤醒一个菜单,选小时和分钟.

同时获取日期和时间 input type="datetime"

已经不兼容了.

同时获取日期和时间 input type="datetime-local"

datetime的是UTC时间格式(世界标准时间),而datetime-local是本地时间.所以更多人用的是datetime-local.

input type="range"

用滑动的方式选择一个值.range有独特的属性min和max,还决定滑杆两边的极值.如果省略了min和max,极值默认为1和100.

input type="search"

与text差不多,只在文本框末端多了一个清空用的小×.

input type="color"

弹出一个颜色选择控件,选取一个颜色.可以用来做在线文本编辑器之类的.

表单属性

autocomplete属性

form域或input域应该拥有自动完成功能,就是在提交过一次后,浏览器保存提交过得信息,下次点击输入框会给出提示能直接填写.

form autocomplete="on" 关闭"off" 默认是on

autofoucs属性

autofocus="autofocus" 页面加载完成后,光标在哪.适用于所有input.

multiple属性

multiple="multiple" 规定输入域中可选择多个值.适用于email和file.

在email中可以通过一个逗号,来分割多个email地址.传给后端时能自动分为数组.

PS:应该是在input中适用于email和file,但在select也是能用上的,成为下拉多选菜单.

placeholder属性

placeholder很好用.

input requirde

requirde="requirde" 表单提交时不得为空,email还得满足一个初级验证

十分简单的验证,所以到底还是得用JS自己写一个.

链接属性

link的sizes属性

link引入icon(图标文件),type="image/gif"时,sizes给出icon的大小.就是浏览器地址栏上的那个小图标,比如百度的熊爪这样.

这样可以根据不同的设备的分辨率给出不同的sizes

link rel="icon" href="icon.gif" type="image/gif" sizes="16*16"

base标签的target属性

base写在head标签里,base只有一个,base决定了页面中所有相对位置从什么地方开始.

换言之,原来的相对位置是以当前页面的位置为base,修改base就能把整个页面的绝对路径转化为相对路径.

base href="http://localhost/" targer="_blank"

那么base中的targer属性就是更改了所有的链接的默认打开方式(原本默认_self).

超链接a的属性media

标明链接需要对什么设备类型进行优化.

media="handheld" 对手持设备进行优化.

media="tv" 对电视设备进行优化.

超链接a的属性hreflang

设置语言,提前告诉浏览器用什么语言,加快渲染时间.

超链接a的属性rel

语义化的表达链接和本文档的关系,比如nofollow:告诉搜索引擎不要追踪此链接,external:告诉搜索引擎这不是本站链接.

script的新属性

defer="defer" 加载完脚本后并不执行,而是等整个页面加载完之后再执行.

async="async" 加载完脚本后立即执行,不用等整个页面都加载完,属于异步执行.
