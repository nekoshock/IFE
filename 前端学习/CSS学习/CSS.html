背景样式
background-color: 背景颜色
background-image: 把图片设置为背景
background-position: 设置背景图片的起始位置
background-attachment: 背景图片是否固定或者虽页面其余部分滚动
background-repeat: 背景图像是否重复或者如何重复
background: 简写属性
背景颜色
background-color:rgb|transparent(透明色)
背景图片
background-image:URL|none
当背景图片和颜色同时设置时,图片总会覆盖掉颜色.当图片无法加载时,颜色就能起作用.
背景图片重复
background-repeat:repeat默认|no-repeat|repeat-x|repeat-y
背景图片显示方式
background-attachment:scroll默认 背景图片随滚动条滚动|fixed 页面其余部分滚动时,背景不会移动
fiexd的位置是根据窗口或者说是网页的位置固定,而不是父元素.
背景图片定位
background-position:百分比|值|top|right|bottom|left|center
可设置两个值,先设置水平位置,后设置垂直位置.如果只写一个参数,另一个默认是居中.
background缩写
各项之间不分顺序.
透明度
opacity
列表样式
list-style-type:
ul: none | disc | circle | squrae
ol: none | decimal | lower-roman | upper-roman | lower-alpha | upper-alpha
列表图片
list-style-image:URL | none
列表项标记位置
list-style-position: inside | outside 默认
样式缩写
list-style: list-style-type list-style-position list-style-image
各项之间不分顺序.
list-style-image总会覆盖掉list-style-type的样式.
浮动 float
会使元素向左或向右移动,只能左右,不能上下.
浮动元素碰到包含框或另一个浮动框,浮动停止.
浮动元素之后的元素将围绕它,之前的不受影响.
浮动元素会脱离标准流.
float: left | right | none ;
清除浮动
clear: none | left | right | both ;
清除浮动的常用方法
1.在浮动元素后面使用一个空元素
2.给浮动元素的父元素添加overflow:hidden;
3.使用:after伪元素为父元素后面添加一个空的元素
