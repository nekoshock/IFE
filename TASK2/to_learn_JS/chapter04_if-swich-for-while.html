JS基础语法 简单操作


if 判断句

if (条件) {条件满足时执行的内容;} 注意等于是==,赋值是=.

if (条件){条件满足时执行的内容;} else{当if条件不满足时执行的内容;}

switch 选择

switch(表达式){
case 值1:
执行代码块1;
break;
case 值n:
执行代码块n;
break;
default:
无匹配值时执行代码块;}

switch会核对表达式的值与哪个case匹配,然后从匹配到的case从上往下执行,直到遇到break跳出.所以要在每项case中加break.

for 循环

for (定义计数变量;循环条件;计数变量改变){循环条件满足时执行代码块;}

定义计数变量时注意是否是新的变量,是的话要用var或let来定义.由于ES6之前没有块级作用域的,if中用var定义的计数变量是会出污染的,要注意.

while 循环

while (循环条件) {循环条件满足时执行代码块;} 循环条件为真时循环.

do {循环语句;} while (循环条件) 与while循环不同的是,do while会先执行一次,然后判断是否再次循环.

break 退出循环

退出循环体,比如用if(特殊情况){break;}这样.

continue 继续循环(叫跳出本次循环可能更合适)

当循环中执行到continue时,会跳出本次循环(continue之后不再更新),本次计数变量会改变.用法如if(特殊情况){continue;}
