JavaScript 计时器

间隔重复执行setInterval()

setInterval(代码,交互时间); interval:间隔,区间;
与其叫计时器,不如说是间隔重复执行.

参数说明：
1. 代码：要调用的函数或要执行的代码串。
2. 交互时间：周期性执行或调用表达式之间的时间间隔，以毫秒计（1s=1000ms）。
返回值: 一个可以传递给 clearInterval() 从而取消对"代码"的周期性执行的值。
调用函数格式(假设有一个clock()函数): setInterval("clock()",1000) 尽量不要用 或 setInterval(clock,1000) 推荐

做一个钟表:

取消clearInterval()

clearInterval(id_of_setInterval);

参数说明:
id_of_setInterval：由 setInterval() 返回的 ID 值。

做一些按钮,操作上面的钟表.

延迟执行 setTimeout();

setTimeout(代码,延迟时间); timeout:暂停时间,超时;在载入后延迟指定时间,然后去执行且仅执行一次。

参数说明：
1. 要调用的函数或要执行的代码串。
2. 延时时间：在执行代码前需等待的时间，以毫秒为单位（1s=1000ms)。

调用函数格式与setInterval()相同.

取消计时器 clearTimeout();

clearTimeout(id_of_setTimeout); 用法与clearInterval()相似.

History对象

history对象记录了用户曾经浏览过的页面(URL)，并可以实现浏览器前进与后退相似导航的功能。

返回上一页面

window.history.back(); 这也相当于window.history.go(-1);

前进至下一个页面

window.history.forward(); 这也相当有window.history.go(1);

返回历史记录中的特定页面

window.history.go(相对位置);

正数为几就是单击"前进按钮"几次,负数为几就是单击"后退按钮"几次.

Location对象

Location:定位;用于获取或设置窗体的URL，并且可以用于解析URL。
