JS的简单教程

为啥呢,JS高级语言设计这本红宝书确实太高级了,是对于JS语言深入了解提升所用书.并不适合初学者,初学还是学方法和实现比较好吧.

JS脚本的引入

在head中或body底部,使用成对script /script标签;或是外部 script src="" /sricontentt 链接进来.

注释方法

//单行注释;/* */为多行注释.

说起来html的注释是方括号括起来的!-- -- ;CSS的注释仅有/* */这一种方法.

简单互动方法

document.write()

将内容直接写在网页上的方法.

alert();

alert:警告;弹出警告框的方法.

confirm

confirm:确认;弹出需要确认的警告框,返回确认(true)或取消(false)的方法.
这里顺道说一下等于号"="的用法,一个=是赋值,两个==是简单比较,三个===是先比较类型,在比较值.

prompt

prompt:v.提示;n.提词;弹出一个需要输入内容的警告框,返回所填内容.

windows.open

打开目标窗口,还可以设置打开窗口的属性. 写法:windows.open("url","_self/_blank/_top","属性1,属性2");

windows.close

关闭目标窗口.写法为:目标窗口变量名.close();

课后练习

当比较 变量==true还不是很好比,类型不一样,环境不一样(严格模式等)比出来什么结果真说不准.

简单DOM操作

获取元素 document.getElementById("ID")

通过ID获取元素,再赋值给变量,就能操纵元素了.

获取或替换HTML元素的内容 Object.innerHTML

Object是被提取出的元素变量名,Object(对象名).innerHTML(属性)=HTML内容(值)这种玩法.

我现在是红的.

只要触发了修改事件,页面就会相应修改的!

获取或替换目标元素的CSS样式 Object.style.属性="新值".

属性就是width,heigth等等;值就是新值.

需要注意的是background-color这类CSS里有-的属性,到JS里就是backgroundColor了,将-去除,-后面的首字母大写.

显示和隐藏 Object.style.display="none"

通过操作display属性来显示或隐藏某内容


获取或替换元素的class类名 Object.className="classname";

通过修改元素的class名,更换元素的class.
