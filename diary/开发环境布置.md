###vscode快捷键
用户修改项:
* beautify file 改为ctrl+b
* markdown preview enhanced 改为alt+b
* open in default browser 改为alt+b
* 删除行 改为ctrl+d
* 将选择添加到下一个查找匹配项 改为ctrl+alt+d
* 视图:切换侧边栏可见性 改为ctrl+alt+b
* 触发建议: ctrl+tab
[常用快捷键](https://www.cnblogs.com/bindong/p/6045957.html)
常用项
+ 代码行缩进 Ctrl+[ 、 Ctrl+]
+ 下方插入一行 ctrl+enter
+ 删除当前行ctrl+d(手动改)
