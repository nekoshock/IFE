JS基础语法


变量

声明变量 var 变量名; 赋值方法 变量名=值;
快速写法: var 变量名1 = 值1, 变量名2 = 值2;

表达式

赋值语句:num=num+1
串表达式(字符串):"I"+"love"+a+"!"
数值表达式:num=num+3*5;
布尔表达式:5>3;num==6;num <60;

操作符

+

可以作为数字加号求和,也能链接字符串(在一方是字符串的时候).

=

赋值.

++ --

num++;自+1
num--;自-1
如同num=num+1;num=num-1;一样.

比较操作符

逻辑操作符 与 或 非

&& 与

两真为真,一假为假.自带断路效果,若左侧为假,右侧便不再执行.

|| 或

一真为真,两假为假.自带短路效果,若左侧为真,右侧便不再执行.

! 非

真变假,假变真.这是单目运算符.

操作符优先级

算术操作符 → 比较操作符 → 逻辑操作符 → "="赋值符号
