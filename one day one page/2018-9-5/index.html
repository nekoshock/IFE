HTML基础语法

列表
ol 有序列表
type = 1 a A i I 等等...
ul 无序列表
type = square circle disc
li 列表项
定义列表
区域dl 项标题dt dd
图像
img src="图片地址" alt="文字替代内容,也可以给搜索引擎看"
绝对路径与相对路径
从盘符开始写地址和相对于页面文件的地址
上一级
../
下一级
/
超链接
a href="链接地址" /a
跳转至站内或站外
空连接:地址写"#"
target属性:
_blank 打开新页面
_self 在当前页面打开页面,此项为默认
title属性:
鼠标放在链接上显示的提示信息
锚链接
在a标签里用name属性(链接命名)给链接定义锚的位置,用a标签herf="#name名"(感觉跟ID有点混?),实现快速跳转.
跳转到不同页面的锚位置
跳转页面位置+名称#锚名称
href="mailto:邮箱地址" 直接打开邮箱
href="文件地址" 下载
表格
table tr td(th)
表格标题: caption
表格的结构:thead tbody tfoot 无论自上而下的写法如何,表格始终按thead tbody tfoot的顺序显示.
table的属性
bgcolor border width align
cellpadding 单元格和内容的空白
cellspacing 单元格之间的空白
frame 外部边框样式
rules 内部边框样式
tr的属性
bgcolor align水平对齐方式 Valign垂直对齐方式
td/th的属性
align Valign bgcolor border width height
colspan="" 跨列
rowspan="" 跨行
被跨行占用位置的单元格要删掉
thead tbody tfoot的属性
align Valign
