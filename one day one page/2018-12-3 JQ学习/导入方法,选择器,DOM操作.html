JQ选择器

JQ选择器支持CSS1到CSS3选择器
提取JQ对象的方法总会返回一个类数组的JQ对象集合,如果没有符合的元素那这个集合length为0,不会报错.
基本选择器
ID #id 返回单个元素
类 .class 返回集合元素
元素名 element 返回集合元素
通配符 * 返回集合元素
selector1,selector2,.......,selectorN 将匹配任意选择器的元素合并后一起返回.
层次选择器
后代
$("ancestor descendant") 后代选择器 选取ancestor里所有descendant元素.(子子孙孙)
子元素
$("parent>child") 子元素选择器 只选取parent下的child元素.(仅子代)
同级下一个
$("prev+next") 兄弟选择器 选取紧接在prev元素后同辈的next元素.(仅一个弟弟)
同级下面全部
$("prev~siblings") 全部弟弟选择器 选取所有在prev元素后面的同辈的siblings元素.(全部弟弟)
兄弟选择器等价方法 .next()
$("prev+next") 全等于 $("prev").next("next")
全部弟弟选择器等价方法 .nextAll()
$("prev~siblings") 全等于 $("prev").nextAll("siblings")
过滤选择器
过滤选择器通过特定过滤规则从元素集合中筛选符合规则的元素.
:first 选取第一个元素
如$("div:first") 选取所有div元素中第一个div
:last 选取最有一个元素
如$("div:last") 选取所有div元素中最后一个div
:not(selector) 去除所有与给定选择器匹配的元素
如$("input:not(.myClass)") 选取所有元素名为input且类不是myClass的元素.
:even 选取所有索引是偶数的元素(0,2,4,6...)(即第一个,第三个,第五个...)
如$("input:even") 选取索引是偶数的input元素
:odd 选取所有索引是奇数的元素(1,3,5,7...)(即第二个,第四个,第六个...)
如$("input:even") 选取索引是奇数的input元素
:eq(index) 选取索引等于index的元素
如$("input:eq(1)) 选取索引等于1的input元素
:gt(index) 选取索引大于index的元素
如$("input:gt(1)") 选取索引大于1的input元素
:lt(index) 选取索引小于index的元素
如$("input:lt(1)") 选取索引小于1的元素
:header 选取所有标题元素(h1,h2,h3...)
如$(":header") 选取网页中所有的标题元素
:animated 选取当前正在执行动画的所有元素
如$("div:animated") 选取当前正在执行动画的div元素
:focus 选取当前获取焦点的元素
如$(":focus") 选取当前获取焦点的元素.
内容过滤筛选器
主要体现在它所包含的子元素或者文本内容上.
:empty 选取不包含子元素或者文本的元素
如$("div:empty") 选取不包含子元素(包括文本元素)的空div元素.
仅此项是与CSS共有的选择器,其他三项为JQ独有.
:contains(text) 选在含有text文本的元素
如$("div:contains('我')") 选取包含文本"我"的div元素.
:has(selector) 含有选择器所匹配元素的元素
如$("div:has('.color')") 选取含有.color类子元素的元素.
:parent 选取含有子元素(或者文本元素)的元素
如$("div:parent") 选取含有子元素或者文本元素的div元素.
可见性过滤选择器
根据元素可见和不可见的状态来选择相应元素.
:hidden 选取所有不可见元素
包括display:none visibility:hidden input type="hidden"
:visible 选取所有可见元素
如$("div:visible") 选取所有可见的div元素.
属性过滤器
根据元素的属性来筛选元素.
[attribute] 拥有attribute属性的元素
$("div[attr]") 拥有attr属性的div元素
[attr=value] 选取有属性attr且值为value的元素
如$("div[title=test]") 选取属性title的值为test的div元素
[attr!=value] 选取没有attr属性或者attr属性值不等于value的元素
如$("div[title!=test]") 选取没有title属性或者有title属性但值不等于test的div元素
[attr^=value] 选取属性值value开头的元素
如$("div[title^=test]") 选取title属性值value开头的div元素
[attr$=value] 选取属性值以value结尾的元素
如$("div[title$=test]") 选取title属性值以value结尾的div元素
[attr*=value] 选取属性值含有value的元素
如$("div[title*=test]") 选取title属性值中含有value的div元素
[attr|='value'] 选取属性值等于给定字符串value或者以value为前缀(即value-xxxx)的元素
如$("div[title|='en']") 选取属性title值等于en或者是en-开头的div元素
[attr~='value'] 选取属性值中有以空格分隔的value字符的元素
如$("div[class~='left']") 选取class属性中有left值的div元素.
[attr1][attr2][attrN] 复合筛选,选择满足多个条件的元素
如$("div[id][title$='test']") 选组含有ID属性,且title属性值以test结尾的div元素.
小小的总结一波
[] 拥有属性
[=] 属性值为
[!=] 属性值不为或没有此属性
[^=] 属性值以..开头
[$=] 属性值以..结尾
[*=] 属性值含有..
[|=''] 属性值为字符串..或者以字符串..-开头
[~=''] 属性值有以空格分隔出来的字符串..
多个条件可以联用,缩小范围.
子元素过滤器
:nth-child(index/even/odd/equation)
符合某种条件下的所有元素的集合,nth-child的数字是从1开始的,第一个孩子,第二个孩子等等...
odd 奇数,even 偶数.
与:eq(index)相比,:eq(index)只返回1个,nth-child(index)是所有此类父子搭配下符合index的.
:first-child 选出某个元素,它需是孩子中第一个
做一个比较的例子
如$('ul li:first-child') 返回所有ul元素中第一个li元素的集合.
而$('ul li:first') 则是返回整个页面中第一个ul下的第一个li元素.
:last-child 选出某个元素,它需是孩子中最后一个
类似上条,不赘述了.
:only-child 选出某个是它的父元素的唯一的子元素的元素
$('ul li:only-child') 选出某个li元素,它需是ul的唯一子元素.
:nth-child详细
1. :nth-child(even) 选取每个父元素下索引值是偶数的元素. 等价于:nth-child(2n)
2. :nth-child(odd) 选取每个父元素下索引值是奇数的元素. 等价于:nth-child(2n-1)
3. :nth-child(2) 选取每个父元素下索引值等于2的元素.
4. :nth-child(3n) 选取每个父元素下索引值是3的倍数的元素.(n从1开始)
5. :nth-child(3n+1) 选取每个父元素下索引值是(3n+1)的元素.
插一条 神奇的空格
$("div:first-chlid")和$(div :first-chlid)之间有何区别?
对,前者冒号前没有空格,而后者有.
这下意思可发生了大转变.
前者是选择一个div元素,它需是父元素中第一个孩子.
后者是选择所有div元素下第一个子元素.
可以理解为 这个空格与后面直接跟着的冒号之间,有一个被省略的*通配符.
所以$(div :first-chlid)就是$(div *:first-chlid),翻译为选择所有div下任意第一个子元素.
有没有空格会有很大的不同.

JQ中的DOM操作

查找节点
1.查找元素节点
见上文选择器.
2.查找属性节点 .attr()方法
$("p").attr("属性名","属性值(可选)")
[js]类似整合了setAttribute和getAttribute
创建节点
1.创建元素节点
使用JQuery中的函数工厂$()来完成. $(html)
如var $li_1 = $("<li></li>")
然后将这个$li_1插入到DOM树中就好.
[js]document.createElement("html")
2.创建文本节点
在创建元素节点时将文本写入元素,就自然创建节点了.
如var $li_1 = $("<li>这是文本节点,li是元素节点.</li>")
[js]document.createTextNode(text)
3.创建属性节点
也是在创建元素节点时,一同被创建.
如var $li_2 = $("<li class='left'>"文本节点</li>")
[js]setAttribute("attr","value")
插入节点
1.append() 向每个匹配的元素内部追加内容
2.appendTo() append()的倒写,把追加内容写入匹配元素内部最后.
3.prepend() 向每个匹配的元素内部前置内容
4.prependTo() prepend()的倒写.
5.after() 在每个匹配元素之后追加内容
6.insertAfter() after()的倒写.
7.before() 在每个匹配的元素之前插入内容.
8.insertBefore() before的倒写.
由于元素只有一个位置,所以插入节点不仅可以插入新节点,也可以移动原有节点的位置.
5.删除节点
1.remove()方法
$(ele).remove(selector);
删除ele元素中,满足selector的元素.
返回被删除元素的引用,因此可以再使用被删除的元素.
如var $li = $("ul li:fisrt").remove(); $li.appendTo("ul")
将ul中第一个li元素删除,然后添加回ul的最后.
等价于 $("ul li:first").appendTo("ul")
2.detach()方法
与remove()方法一样,返回被删除的元素.但保留此元素绑定的事件和附加的数据等.
3.empty()方法
清空元素里的内容,包括文本等后代节点. 会留下空的元素自身
4.clone() 复制节点
如var $i_2= $(ul li:first).clone() 复制一个ul下第一个li元素节点
被复制出来的新节点没有原来的绑定事件,但如果clone()方法传递一个参数ture,就能同时复制元素所绑定的事件.
如 $i_2= $(ul li:first).clone(ture)
5.replaceWith()和replaceAll() 替换节点
replaceWith()将所有匹配的元素替换为制定的HTML或者DOM元素.
如$("p").replaceWith("<b>哈哈哈</b>") 将所有的P元素换成<b>哈哈哈</b>
replaceAll()是replaceWith()的倒写.
原绑定的事件会和被替换元素一同消失,需要在新元素上重新绑定事件.
6.wrap() 包裹节点
如果需要将某个节点用其他标记包裹起来,用warp()方法.
如$("p").wrap("<div></div>") 用div包裹每一个P标签.
6.1 wrapAll() 将所有匹配到的元素包裹在同一组标签内.
传入的参数是成对标签.
如<p>一部分</p><p>另一部分</p>
用$("p").wrapAll("<div></div>")
便会得到<div><p>一部分</p><p>另一部分</p></div>
如果被包裹的多个元素间有其他元素,其他元素会被放到包裹元素之后.
6.2 warpInner() 将所有匹配元素的子内容用标记包裹起来.
如$("p").wrap("<div></div>") 结果是<p><div>子内容...</div></p>
7.属性操作
7.1 获取和设置属性
如var p.attr = $("p").attr("title") 获取属性
如$("p").attr("title","test") 设置单个属性
也可一次性设置多个属性
如$("p").attr("title":"test","class":"left")
用名:值形式的对象作为参数.
7.2 removeAttr() 删除属性
如$("p").removeAttr("title") 删除所有p元素的title属性.
prop()
获取元素集中第一个元素的属性值.
removeProp()
为匹配的元素删除设置的属性,
8.样式操作
用attr("class","....")来修改类名.
8.1 addClass() 追加类名
如$("p").addClass("left") 为所有p元素追加left类名,且不影响原有类名.其余按照CSS的覆写顺序进行.
8.2 removeClass() 移除类名
如$("p").addClass("left") 为所有P元素移除left的类名,其余类名不受影响.
需要一次移除多个类名的话有快捷方法: 如$("p").addClass("left float");,将多个类名用空格隔开.
如果没有参数,就会移除所有类名.
9. toogle() 切换样式
$('ele').toggle(function() {
//③
} , function() {
//④
})
toggle()的作用是在两个函数中交替切换.
9.1 toggleClass() 类名若存在则删除他,类名若不存在则添加他.
10. hasClass() 是否有某个类(某个)样式
如$("p").hasClass("another"); 如果p的class里有another则返回ture,如果没有就返回false.
实际上hasClass是is的换皮,为的是增加可读性.
所以$("p").hasClass("another");和$("p").is("another");是等价的.
获取和设置HTML\文本\值
1.html()方法
类似于JS的innerHTML属性.
如 <p><b>文本内容</b></p>
$("p").html();
会返回<b>文本内容</b>
设置方法:html("设置内容"); 将设置内容作为参数传给html()
2.text()方法
类似于JS的innerText属性. 但innerText有浏览器兼容问题,text()没有
用法同上.
3.val()方法
类似于JS的value属性.
返回文本框,下拉列表或是单选框的元素的值,如果元素是多选,返回一个所有选择的值的数组.
除了获取和设置value,val()方法还能选中select(下拉列表项),checkbox(多选框)和radio(单选框)的选项.
方法如下:$(select,checkbox,radio等).val("选中项的值") 若选中多选,就以数组作为参数["选中1","选中2"...]
如$(":checkbox").val(["check2","check3"])
也可以用attr()修改选项的selected或checked值为ture,来选中.
